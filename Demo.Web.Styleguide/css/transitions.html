Transition Basics

transition: property duration timing-function delay

Transition one property

Only 'color' will transition, all other properties are ignored

Anything that has an intermediate value (ie has :halfway) can be transitioned 

            
              // e.g.
              a {
                transition: color 2s ease-in 50ms;
              }
Hover over me

Transition all properties 

            
              transition: all 2s ease-in 50ms;
Hover over me

Transition transforms

Hover over image to start 

            
              transition: all 3s ease-in 50ms;
               then on hover,
              transform: rotate(360deg) scale(2);
