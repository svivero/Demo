BEM

NOTE: Dosn't use descendent selectors.

            
              ul> class="menu" // block
                li> class="menu__item" // element
                li> class="menu__item menu--hidden" // with element modifier

BEM - Block

A logically and functionally independent page component.

Blocks being independent allows for their re-use.

Can be nested with other blocks.

Block Naming Convention

Use only classes, lowercase letters and one hyphen separate words. 

            
              .block-name
              .menu

BEM - Element

A element is part of a block. It is a child element of the block and cannot be used on its own. E.g. the tabs below cannot be used outside their Block.

Element Naming Convention

Delimited by a double underscore. 

            
              .block-name__elem-name
              .menu__item

BEM - Modifier

Defines the appearance and behavior of a block or element.

Modifiers can be used to change the block's theme or style the component without modifying other unrelated modules.

Modifier Naming Convention

Modifiers should identify which block it belongs to, followed by a modifier name, separated by a single underscore.

Or use the alternative naming scheme, two hyphens instead of an underscore. 

            
              .block-name__elem-name_mod-name
              .block-name__elem-name--mod-name (alternative)
              .menu__item_hidden
              .menu__item--hidden (alternative)
