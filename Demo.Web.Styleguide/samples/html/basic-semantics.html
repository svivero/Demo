Header: Introductory information
Nav: Group navigation links
Main: Primary page content

Article: Independent item of content

Content that make sens on its own e.g.

  • Magazine article
  • Newspaper article
  • Blog post

See: HTML5 Element Flowchart

Section: Content included in the document outline

NOTE: A section should include a heading

Div: Used if an element is mainly used for styling

Aside: Separate but related to the parent content

Common scenarios are:

  • Content usually contained in a sidebar but not based on visual style
  • Pull quotes
  • Ads
Footer: Metadata about its section ie metadata or copyright
